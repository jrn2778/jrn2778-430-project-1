Developers

Access the API with the following endpoints:

  • /good-action (Get a single, random good action)
  • /good-actions (Get all the good actions)
  • /good-actions?limit=3 (Get a certain amount of random good actions)
  • /good-actions?tags=environment&tags=animals (Get any good action that has at least one of the given tags)
